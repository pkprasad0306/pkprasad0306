## 🚀 Featured Projects

<details>
<summary><b>🛰️ SARFormer-GAN — Satellite Cloud Removal (ISRO BAH 2026)</b></summary>
<br/>

A novel deep generative architecture developed for ISRO's Bharatiya Antariksh Hackathon (Problem Statement BAH-DS-3), engineered to remove cloud occlusion from LISS-IV satellite imagery using a hybrid Swin-Transformer + GAN pipeline.

| Aspect | Detail |
|---|---|
| **Stack** | PyTorch, Swin-Transformer, PatchGAN, Python, NumPy |
| **Scale** | High-resolution multi-band satellite imagery (LISS-IV) |
| **Performance** | Cross-attention fusion for spatial-spectral consistency |
| **Security** | Isolated offline inference pipeline, no external data leakage |
| **Impact** | Official hackathon submission with full 10-slide ISRO-template deck and 7-page technical report |
| **Repository** | [View on GitHub](https://github.com/pkprasad0306) |

**Architecture:** Swin-Transformer based generator for long-range spatial dependency modeling, combined with cross-attention fusion layers to merge multi-temporal satellite bands, paired with a PatchGAN discriminator for high-fidelity, artifact-free reconstruction of cloud-occluded regions.

</details>
